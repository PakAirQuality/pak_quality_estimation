# Daily gridded PM2.5 predictions, 2024–2025

Daily surface PM2.5 estimates over Pakistan on a fixed **0.1° (~10 km) grid**, produced by
the support-aware LightGBM model in this repository. This is the gridded product referenced
in the manuscript's Code and data availability statement.

## Contents

For every day from **2024-01-01 to 2025-12-31**:

| File | Description |
|---|---|
| `pm25_YYYY-MM-DD.tif` | Raw daily PM2.5 field (GeoTIFF, µg m⁻³), masked to Pakistan |
| `pm25_YYYY-MM-DD_gaussian_smoothed.tif` | Gaussian-smoothed field for map tiling/display |
| `pm25_YYYY-MM-DD.json` *(or sidecar)* | Per-day feature-coverage diagnostics (feature count, NaN fraction, per-stage coverage for AOD/MET/TROPOMI) |

~1,462 GeoTIFFs + ~731 per-day metadata records (~49 MB total).

## Format

- **CRS / grid:** geographic (lat/lon), 0.1° spacing over the Pakistan domain
  (~141 × 175 cells; ~9,682 in-country cells).
- **Units:** µg m⁻³ (daily mean surface PM2.5).
- **Reading:** any GeoTIFF reader, e.g. `rasterio.open(path).read(1)`.

## Provenance & caveats

Generated by the archive-quality support-aware estimator (see repository `README.md` /
`MODEL_CARD.md`). Values are daily grid-cell means, not site-level measurements, and should be
interpreted at neighbourhood scale (see the manuscript's limitations). Predictions in
unmonitored cells rely on the satellite/meteorology/composition pathway (no local PM history).
